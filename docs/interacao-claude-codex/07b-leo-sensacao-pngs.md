## Como preencher

Use uma das três palavras:

| Marca | Significado |
|---|---|
| `ok` | Pode seguir como base, sem mexer agora |
| `fraco` | Funciona, mas não está no padrão V5 |
| `estranho` | Algo incomoda, quebra confiança ou parece bug |

Se quiser, escreva só na coluna `Observação Léo`. Não precisa justificar tudo.
